# ShiroMail 功能与页面布局优化方案

> 基于当前项目完整分析，覆盖功能缺失、交互流程、页面布局、组件拆分、响应式体验等维度。

---

## 一、Landing Page（公共首页）

### 1.1 当前状态

- 已有 Hero 区域、功能卡片、统计徽章、Checklist 展示
- 使用 PublicUI 组件系统（PublicSection / PublicFeatureCard / PublicStatBadge）
- 有 ambient 动画背景主题

### 1.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| L1 | 添加产品截图/动画演示区 | 高 | Hero 下方插入交互式 Demo 或截图轮播，降低用户认知成本 |
| L2 | 增加 FAQ 折叠面板 | 中 | 使用 Accordion 组件，解答常见问题（隐私、数据保留、API 限制） |
| L3 | 添加 Pricing 区块或独立页 | 高 | 当前 `/pricing` 路由存在但内容为空壳，需完善定价卡片对比 |
| L4 | 用户评价/案例展示 | 低 | 社会证明区块，可后期补充 |
| L5 | CTA 按钮优化 | 中 | Hero 区 CTA 增加次要操作（如"查看 API 文档"），双按钮布局 |
| L6 | 页脚信息完善 | 低 | 补充 Legal、Privacy Policy、Terms 链接 |

## 二、用户控制台 Dashboard

### 2.1 当前状态

- 路由 `/console/dashboard`，展示基础统计指标
- 使用 WorkspaceMetric 卡片展示数字
- 无图表、无活动流、无快捷操作

### 2.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| D1 | 7天/30天收件趋势图 | 高 | 使用 recharts 或 chart.js，折线图展示每日收件量 |
| D2 | 最近邮件活动流 | 高 | 时间线组件，展示最近 10 条收件记录（发件人、主题、时间） |
| D3 | 快捷操作卡片 | 中 | "创建邮箱"、"添加域名"、"查看 Webhook 日志" 快捷入口 |
| D4 | 邮箱容量/配额可视化 | 中 | 进度条展示各邮箱已用/总配额 |
| D5 | 系统通知横幅 | 低 | 管理员发布的公告在 Dashboard 顶部展示 |
| D6 | 引导式新手流程 | 高 | 首次登录检测，展示 Stepper 引导（创建域名→验证 DNS→创建邮箱→测试收件） |

## 三、邮箱与邮件页面

### 3.1 当前状态

- `mailboxes-page.tsx`（1113 行）：邮箱列表 + 创建/编辑对话框
- 邮件查看通过独立路由，无收件箱视图
- 无搜索、无批量操作、无邮件转发

### 3.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| M1 | 收件箱视图 | 高 | 新增 `/console/mailboxes/:id/inbox` 路由，左侧邮件列表 + 右侧预览面板（Master-Detail 布局） |
| M2 | 邮件搜索 | 高 | 支持按发件人、主题、日期范围搜索，前端 Combobox + 后端全文索引 |
| M3 | 批量操作 | 中 | 多选邮件后支持批量删除、标记已读、导出 |
| M4 | 邮件转发规则 | 中 | 每个邮箱可配置转发地址，收件后自动转发到外部邮箱 |
| M5 | 邮件详情增强 | 中 | 附件预览（图片内联、PDF 预览）、原始邮件头折叠展示、下载 .eml |
| M6 | 未读标记与计数 | 高 | 邮箱列表显示未读数 Badge，侧边栏邮箱项也展示 |
| M7 | 组件拆分 | 高 | 将 1113 行拆分为：MailboxList、MailboxCreateDialog、MailboxEditDialog、MailboxCard |
| M8 | 空状态引导 | 中 | 无邮件时展示引导卡片："发送测试邮件到此地址试试" |

## 四、域名与 DNS 管理

### 4.1 当前状态

- `dns-page.tsx`（3062 行）：最复杂页面，包含域名验证、DNS 记录展示、状态检查
- 功能完整但代码臃肿，维护困难

### 4.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| N1 | 组件拆分（必须） | 高 | 拆分为：DomainList、DomainDetail、DnsRecordTable、DnsVerifyWizard、DnsStatusBadge（每个 < 300 行） |
| N2 | DNS 验证向导 | 中 | 步骤化引导：选择 DNS 提供商 → 展示需添加的记录 → 一键验证 → 结果反馈 |
| N3 | 复制按钮优化 | 低 | DNS 记录值旁添加一键复制，带 Toast 反馈 |
| N4 | 自动刷新状态 | 中 | DNS 验证中的域名每 30s 自动轮询状态，验证通过后停止 |
| N5 | 域名健康度面板 | 低 | 汇总展示 MX/SPF/DKIM/DMARC 配置完整度评分 |

## 五、Webhook 管理

### 5.1 当前状态

- 支持创建/编辑/删除 Webhook
- 新增了 delivery log 后端，但前端尚未展示

### 5.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| W1 | 投递日志页面 | 高 | 新增 `/console/webhooks/:id/logs` 路由，表格展示投递记录（状态码、延迟、错误信息） |
| W2 | 手动重试按钮 | 中 | 失败的投递记录支持手动重试 |
| W3 | Webhook 测试 | 中 | 创建/编辑时支持发送测试 Payload，实时展示响应 |
| W4 | 事件过滤器 | 低 | 支持选择触发事件类型（新邮件、邮件删除等） |
| W5 | 签名验证说明 | 低 | 页面内嵌文档，说明如何验证 Webhook 签名 |

| A7 | 数据导出 | 中 | 支持导出用户列表、邮件统计为 CSV |

## 七、布局与导航优化

### 7.1 当前状态

- PublicShell：顶部导航 + 内容区
- ConsoleShell：可折叠侧边栏 + 顶部栏 + 内容区
- 响应式通过 Tailwind 断点 + `use-mobile` hook 处理
- 无面包屑、无键盘快捷键、无 Skip-to-content

### 7.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| Y1 | 面包屑导航 | 中 | Console 页面顶部添加面包屑，展示当前位置层级 |
| Y2 | 移动端抽屉导航 | 高 | 小屏幕下侧边栏改为 Sheet 抽屉，点击遮罩关闭 |
| Y3 | Skip-to-content | 中 | 无障碍：Tab 首焦点为跳转到主内容区的隐藏链接 |
| Y4 | 键盘快捷键 | 低 | `Ctrl+K` 全局搜索、`G then D` 跳转 Dashboard、`G then M` 跳转邮箱 |
| Y5 | 侧边栏邮箱快捷列表 | 中 | 侧边栏底部展示最近使用的 3 个邮箱，点击直接进入收件箱 |
| Y6 | 页面切换动画 | 低 | 路由切换时添加 fade/slide 过渡，使用 Framer Motion 或 View Transitions API |
| Y7 | 响应式表格 | 中 | 移动端表格改为卡片列表布局，避免横向滚动 |
| Y8 | 顶部栏增强 | 低 | 添加通知铃铛图标（未来通知中心入口）、快速切换邮箱下拉 |

## 八、交互与反馈优化

### 8.1 当前状态

- 使用 NoticeBanner 做操作反馈
- AlertDialog 做危险操作确认
- 无 Toast 通知、无加载骨架屏、无乐观更新

### 8.2 优化项

| # | 优化点 | 优先级 | 说明 |
|---|--------|--------|------|
| X1 | Toast 通知系统 | 高 | 替代部分 NoticeBanner，操作成功/失败用 Toast 短暂提示（shadcn/ui Sonner） |
| X2 | 骨架屏加载 | 中 | 列表页、详情页加载时展示 Skeleton 占位，替代空白或 Spinner |
| X3 | 乐观更新 | 中 | 删除邮件、切换状态等操作先更新 UI 再等后端确认，失败时回滚 |
| X4 | 表单验证增强 | 中 | 实时校验 + 错误信息国际化，使用 zod + react-hook-form |
| X5 | 确认对话框统一 | 低 | 所有危险操作统一使用 AlertDialog，文案模板化 |
| X6 | 拖拽排序 | 低 | Webhook 列表、邮箱列表支持拖拽调整顺序 |
| X7 | 无限滚动 | 中 | 邮件列表使用虚拟滚动（tanstack-virtual），替代分页 |

## 十、组件拆分路线图

针对当前超大文件的拆分建议：

```
dns-page.tsx (3062 行)
├── DomainListSection.tsx        — 域名列表与筛选
├── DomainDetailPanel.tsx        — 单个域名详情
├── DnsRecordTable.tsx           — DNS 记录表格
├── DnsVerifyWizard.tsx          — 验证向导步骤
├── DnsStatusIndicator.tsx       — 状态徽章与刷新
└── hooks/use-dns-verification.ts — 验证轮询逻辑

admin-settings-page.tsx (1165 行)
├── GeneralSettingsTab.tsx       — 站点名称、URL 等
├── SmtpSettingsTab.tsx          — SMTP 配置
├── SecuritySettingsTab.tsx      — 密码策略、注册开关
├── InboundPolicyTab.tsx         — 附件限制、可执行文件拦截
└── hooks/use-settings-form.ts   — 表单状态管理

mailboxes-page.tsx (1113 行)
├── MailboxList.tsx              — 邮箱卡片列表
├── MailboxCreateDialog.tsx      — 创建对话框
├── MailboxEditDialog.tsx        — 编辑对话框
├── MailboxCard.tsx              — 单个邮箱卡片
└── hooks/use-mailbox-actions.ts — CRUD 操作封装
```
